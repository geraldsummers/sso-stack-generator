Disposable Workspaces

Provision SSH-first agent workspaces on the isolated labware Docker runtime. TTYD shell access is Keycloak-gated and follows the workspace lifecycle. Notebook access is optional, path-based, and mounted against the same workspace home volume. Each workspace also gets controller-issued knowledge access for search and exact raw-document retrieval.
